    CORDEX-CMIP7 Archiving Specifications¶

    Overview¶

    This section will provide the archiving specifications for CORDEX-CMIP7 simulations.

    Coming Soon

    CORDEX-CMIP7 archiving specifications are currently under development. Please check back later for updates.

    Status¶

    The CORDEX-CMIP7 archiving specifications are being developed by the CORDEX Task Team on CORDEX-CMIP7 Protocol and Infrastructure (TTPI), in alignment with:

    • CMIP7 data standards and conventions
    • CORDEX modeling groups
    • ESGF infrastructure updates
    • CF Conventions updates

    Anticipated Updates¶

    The CORDEX-CMIP7 specifications will likely include:

    • Updated DRS elements aligned with CMIP7
    • New or modified global attributes
    • Enhanced metadata requirements
    • Updated file format specifications
    • New controlled vocabularies

    Stay Informed¶

    For the latest information on CORDEX-CMIP7 development:

    • Monitor the archive-specifications repository
    • Watch for announcements on the CORDEX website
    • Subscribe to repository notifications

    Current Specifications¶

    For ongoing work, please use the CORDEX-CMIP6 Archiving Specifications.
