# CORDEX-CMIP6 Archiving Specifications

This repository collects [issues](https://github.com/WCRP-CORDEX/archive-specifications/issues) related to the _CORDEX Archiving Specifications_, which are maintained here as mkdocs sources and officially released in Zenodo.

## Dynamical downscaling (CORDEX-Domain)
Different versions of the specifications have been released. Please, always refer to the [latest version](https://doi.org/10.5281/zenodo.10961068) when processing your data.
