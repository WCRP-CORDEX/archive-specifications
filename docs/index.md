# CORDEX-CMIP6 Archiving Specifications

This site provides the archiving specifications for CORDEX-CMIP6 simulations.

The specifications define the Data Reference Syntax (DRS) elements necessary for post-processing CORDEX-CMIP6 simulations and publishing them on the Earth System Grid Federation (ESGF).
This includes file and directory naming conventions, global attributes, and ESGF Search Facets Mapping.
